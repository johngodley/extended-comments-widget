== Description ==

A widget that shows a section of comment text along with the author name.

* Show extended comment information
* Configure the number of comments used
* Exclude certain authors from the list (such as yourself)

== Installation ==

The plugin is simple to install:

1. Download `extended-comments-widget.zip`
1. Unzip
1. Upload html-purified directory to your `/wp-content/plugins` directory
1. Go to the plugin management page and enable the plugin
1. Add the widget to your blog via the Appearance => Widgets menu

You can find full details of installing a plugin on the [plugin installation page](http://urbangiraffe.com/articles/how-to-install-a-wordpress-plugin/).
